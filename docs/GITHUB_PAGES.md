# GitHub Pages Setup

1. Push this repository to GitHub.
2. Open repository **Settings**.
3. Go to **Pages**.
4. Select branch: `main`.
5. Select folder: `/root`.
6. Save.
7. Open the published URL after GitHub Pages finishes building.

## localStorage warning

Progress is tied to the exact browser origin. These are different storage locations:

- `file:///.../index.html`
- `https://username.github.io/stephens-simple-kanji/`
- `https://custom-domain.example/`

Export a backup before changing URLs or domains, then import it at the new location.
